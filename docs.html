Here's a short tutorial on how to add liquidity, swap tokens, and remove liquidity on the PepeX platform.



1. Added Liquidity:


Step 1: Go to the PepeX DEX page. Connect your wallet, after the wallet is successfully connected.



Step 2: Click "Liquidity" or "Pool" Click add Liquidity.



Step 3: Select the token pair you want to add liquidity to. For example, you can choose the BNB/PEPEX pair.



Step 4: Enter the number of tokens of both assets you want to add as liquidity. Make sure you have an equal number of each token in the pair.



2. Swap Tokens:


Step 1: Return to the PepeX DEX page.



Step 2: Click "Swap".



Step 3: Select the token pair you want to exchange. For example, you can choose BNB/PEPEX.



Step 4: Enter the number of tokens you want to exchange and select the asset you want to receive as a result of the exchange.



Step 5: View transaction summary and confirm. The transaction will be executed immediately.



3. Removing Liquidity:


Step 1: Go to the PepeX DEX page and click "Liquidity" or "Pool".



Step 2: Select the "Remove Liquidity" option.



Step 3: Select the token pair from the liquidity you want to delete.



Step 4: Enter the number of LP tokens you want to delete (or all of them).



Step 5: Confirm your transaction. Your liquidity will be removed and the assets you added will be returned to you.
