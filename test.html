Introduction

What PepeX?

PepeX is a meme coin and decentralized exchange (DEX) that allows users to exchange various crypto assets without a third party and is non-custodial. utilizes Automated Market Makers (AMM) technology to adjust between 2 parties who want to exchange crypto assets. In AMM, crypto asset exchanges utilize a pool of liquidity pools filled by users. Users who store their crypto assets in liquidity pools will get incentives.

All real time transactions between users and the Platform, PepeX runs using smart-contracts built on several chains that support an Ethereum Virtual Machine-based network, PepeX is built from DEX base code which allows users to transact in a decentralized manner.
