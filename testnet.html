Testnet Tutorial



1. If you don't have OPBNB Coins yet:


a. Grab the faucet at: https://testnet.bnbchain.org/faucet-smart

b. Bridge to OPBNB: https://opbnb-testnet-bridge.bnbchain.org/deposit

If you already have OPBNB coins, this is optional.



2. Swap on PepeX:


a. Go to the PepeX DEX page: https://pepex.sbs/dex



b. connect a Metamask wallet or Trust Wallet, using the OPBNB testnet network.

• Additional RPC if you have not added the OPBNB testnet RPC.

• Go to settings -> networks -> add network page.

• Select add manual network and enter the network information.

Network Name: opBNB Testnet
RPC URL: https://opbnb-testnet-rpc.bnbchain.org
ChainID: 5611
Symbol: tBNB
Explorer: https://opbnbscan.com/



Or add in a simple way: https://chainlist.org/chain/5611

c. Select the BNB/PEPEX token pair, enter the amount and click "Swap".





3. Added Liquidity:


a. Select the "Pool" and "Add Liquidity" page options.



b. Select BNB/PEPEX pair, enter the number of tokens.



c. Click "Add Liquidity" or "Supply" and confirm the transaction.
